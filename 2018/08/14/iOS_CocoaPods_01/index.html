CocoaPods 项目集成拾遗

公司项目组件化有一段时间了，根据自己的零星笔记，总结一下 CocoaPods 在组件化过程中的一些使用。

参考：CocoaPods 官方文档

CocoaPods 实用

安装升级 CocoaPods

/usr/local/bin ：gem 安装路径。

CocoaPods 常用路径

  • ~/.cocoapods ：本地 spec 库（索引库）。

~/.cocoapods/repos/master 是 origin https://github.com/CocoaPods/Specs.git

  • ~/Library/Caches/CocoaPods ：本地库缓存。

Podfile

  • 使用 CocoaPods 常用命令。
  • 配置 CocoaPods 的 Podfile。

对单个 pod 库可以单独指定下载源，是加载本地、代码仓库的 git 源、索引库。

为 Podfile 指定索引库下载源。

  • Podfile 的一些钩子 hook。

库文件已经下载，安装之前。可以用来删除不同第三方SDK库中冲突文件。比如多个 支付宝SDK 和 网易云信 NIMSDK 都依赖 Openssl/libcrypto.a 和 Openssl/libssl.a，就可以在将要安装的时候，删除其中一个就好。

生成 Xcode project ，在写入磁盘之前。在这个钩子中可以配置 Build Setting 的一些配置。 比如配置 SYSTEM_HEADER_SEARCH_PATHS （search path）。
